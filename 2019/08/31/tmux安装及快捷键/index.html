tmux安装及快捷键

2019-08-31 PV:

安装：sudo apt install tmux 就这么简单

快捷键

Config

快捷键 说明
C-b
e 打开Vim编辑器编辑.tmux.conf.local
r 重载配置

Misc

快捷键 说明
x或Esc 退出copy-mode模式
C-l 清屏
? 打开帮助信息
m 使能或禁止鼠标
: 打开命令行
i 显示tmux信息
~ 显示message
t 时钟模式

Session

快捷键 说明
C-c 创建新session
$ 重命名session
( 上一个session
) 下一个session
s 选择session
C-f 使用名称打开其他session
d 离开session, 回到命令行用tmux list-session查看SessionNumber，用tmux attach -d -t SessionNumber重新进入session
C-z 挂起 session

Window

快捷键 说明
c 创建新window
, 重命名window
f 查找window
w 选择window
. move window
o Rotate window
C-h 打开上一个window
C-l 打开下一个window
1~9 打开第N个window
‘ 打开第N个window
Tab 打开最后一个active的window
& 关闭当前窗口

Pane

快捷键 说明
-或” 水平分隔
_或% 垂直分隔
x 关闭pane
j 转移到下边
k 转移到上边
h 转移到左边
l 转移到右边
J 下边伸长
K 上边伸长
H 左边伸长
L 右边伸长
+ 最大化和复原(会创建新window，旧window变为dead)
z 最大化和复原
q 显示pane
; 打开最后一个pane
o 循环切换pane
<或{ 交换pane
>或} 交换pane
! 单独把pane放到新窗口
Space 切换窗口布局

Copy

快捷键 说明
Enter 进入复制模式
Esc 进入复制模式
[ 进入复制模式
y 复制
] 粘贴最后一个复制
p 粘贴最后一个复制
P 选择粘贴, j/k上下选择，d删除
b或# 列出buffer
= 选择-Z buffer
